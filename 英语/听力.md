# 核心概念

### 1.视听一致

听到的信息和看到的信息越一致越容易正确（简单题）

- 某个选项的关键信息与音频内容完全一致
- 某一个选项的关键信息在音频中出现次数最多

### 2.选取关键词

- 纵向审题：排除相同概念（排除每个选项长得一样的概念）
- 横向划词
  - 划实不划虚（v，n，**否定和消极的词**）
  - 组合概念：名+名/动+名/形+名

### 3.同转概念

- 同根词
- 近义词
- 短语/概念替换
- 宽泛->具体

### 4.正确及其错误选项的特点

- 正确选项
	1. 选项与音频内容一致
	2. 符合题目描述
	3. 通过逻辑引导答案
		1. 因果（therefore，thus，thereby）
		2. 转折
		3. 强调
		4. 举例
		5. 疑问
		6. 感情色彩（赞同否定，强调个人观点，迟疑或者由于）
- 错误选项
	1. 直接被否定
	2. 反义表述
	3. 部分信息没有被提及
	4. 选项杂糅
