# 基本积分公式

以下 10 组公式，要牢记。

### ① 幂函数

- $\int x^k dx = \frac{1}{k+1} x^{k+1} + C, \quad k \neq -1$
- $\int \frac{1}{x^2} dx = -\frac{1}{x} + C$
- $\int \frac{1}{\sqrt{x}} dx = 2\sqrt{x} + C$

### ② 对数相关

- $\int \frac{1}{x} dx = \ln|x| + C \quad \longrightarrow (\ln|x|)' = \frac{1}{x}$

### ③ 指数函数
- $\int e^x dx = e^x + C$
- $\int a^x dx = \frac{a^x}{\ln a} + C, \quad a > 0 \text{ 且 } a \neq 1 \quad \longrightarrow (a^x)' = a^x \ln a$

### ④ 三角函数
- $\int \sin x dx = -\cos x + C$
- $\int \cos x dx = \sin x + C$
- $\int \tan x dx = -\ln|\cos x| + C$
- $\int \cot x dx = \ln|\sin x| + C$
- $\int \frac{dx}{\cos x} = \int \sec x dx = \ln|\sec x + \tan x| + C$
- $\int \frac{dx}{\sin x} = \int \csc x dx = \ln|\csc x - \cot x| + C$
- $\int \sec^2 x dx = \tan x + C$
- $\int \csc^2 x dx = -\cot x + C$
- $\int \sec x \tan x dx = \sec x + C$
- $\int \csc x \cot x dx = -\csc x + C$

### ⑤ 反正切相关
- $\int \frac{1}{1+x^2} dx = \arctan x + C$
- $\int \frac{1}{a^2+x^2} dx = \frac{1}{a} \arctan \frac{x}{a} + C \quad (a > 0)$

### ⑥ 反正弦相关
- $\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C$
- $\int \frac{1}{\sqrt{a^2-x^2}} dx = \arcsin \frac{x}{a} + C \quad (a > 0)$

### ⑦ 根号下的对数形式
- $\int \frac{1}{\sqrt{x^2+a^2}} dx = \ln(x + \sqrt{x^2+a^2}) + C \quad (\text{常见 } a=1)$
- $\int \frac{1}{\sqrt{x^2-a^2}} dx = \ln|x + \sqrt{x^2-a^2}| + C \quad (|x| > |a|)$

### ⑧ 平方差形式
- $\int \frac{1}{x^2-a^2} dx = \frac{1}{2a} \ln \left| \frac{x-a}{x+a} \right| + C$
- $\int \frac{1}{a^2-x^2} dx = \frac{1}{2a} \ln \left| \frac{x+a}{x-a} \right| + C$

### ⑨ 根号下的平方和/差
- $\int \sqrt{a^2-x^2} dx = \frac{a^2}{2} \arcsin \frac{x}{a} + \frac{x}{2} \sqrt{a^2-x^2} + C \quad (a > |x| \geq 0)$

### ⑩ 三角函数平方及其他
- $\int \sin^2 x dx = \frac{x}{2} - \frac{\sin 2x}{4} + C \quad \left( \sin^2 x = \frac{1-\cos 2x}{2} \right)$
- $\int \cos^2 x dx = \frac{x}{2} + \frac{\sin 2x}{4} + C \quad \left( \cos^2 x = \frac{1+\cos 2x}{2} \right)$
- $\int \tan^2 x dx = \tan x - x + C \quad (\tan^2 x = \sec^2 x - 1)$
- $\int \cot^2 x dx = -\cot x - x + C \quad (\cot^2 x = \csc^2 x - 1)$

# 函数奇偶性 (对称性)

在定积分计算中，利用对称区间 $[-a, a]$ 的奇偶性可以极大地简化计算。

### 1. 常见奇函数 (满足 $f(-x) = -f(x)$，关于原点对称)
- **幂函数类**: $x, x^3, x^5, \dots, x^{2n-1}$；以及 $x^{1/3}$ (三次根号)
- **三角函数类**: $\sin x, \tan x, \cot x, \csc x$
- **反三角函数类**: $\arcsin x, \arctan x$
- **对数复合类 (高频考点)**:
    - $f(x) = \ln(x + \sqrt{x^2 + 1})$ (反双曲正弦 $\text{arsinh } x$)
    - $f(x) = \ln \frac{1-x}{1+x}$
- **双曲函数类**:
    - $f(x) = \sinh x = \frac{e^x - e^{-x}}{2}$
    - $f(x) = \tanh x = \frac{e^x - e^{-x}}{e^x + e^{-x}}$

### 2. 常见偶函数 (满足 $f(-x) = f(x)$，关于 $y$ 轴对称)
- **幂函数类**: $x^2, x^4, \dots, x^{2n}$；以及常数 $C$
- **三角函数类**: $\cos x, \sec x, |\sin x|, |\cos x|$
- **其他**: $e^x + e^{-x}, \text{cosh } x$

### 3. 奇偶性组合判断准则
- **加减法**:
    - 奇 $\pm$ 奇 = 奇
    - 偶 $\pm$ 偶 = 偶
- **乘除法 (口诀: 同偶异奇)**:
    - 奇 $\times$ 奇 = **偶**
    - 偶 $\times$ 偶 = **偶**
    - 奇 $\times$ 偶 = **奇**
- **复合函数 $f(g(x))$**:
    - **内偶则偶**: 只要内层 $g(x)$ 是偶函数，复合结果就是偶函数。
    - **外偶则偶**: 只要外层 $f(x)$ 是偶函数，复合结果就是偶函数。
    - **全奇才奇**: 只有当内外层**都是**奇函数时，结果才是奇函数。

### 4. 定积分对称性结论
若积分区间为 $[-a, a]$：
- 若 $f(x)$ 为奇函数，则 $\int_{-a}^{a} f(x) dx = 0$
- 若 $f(x)$ 为偶函数，则 $\int_{-a}^{a} f(x) dx = 2\int_{0}^{a} f(x) dx$

# 常见定积分结论

### 1. 周期性 (Periodicity)
设 $f(x)$ 为连续的周期函数，周期为 $T$：
- 对于任意实数 $a$：$\int_{a}^{a+T} f(x) dx = \int_{0}^{T} f(x) dx$
- 即：在长度为一个周期的区间上的定积分，与区间的起点位置无关。

### 2. 区间再现公式
设 $f(x)$ 为连续函数：
- $\int_{a}^{b} f(x) dx = \int_{a}^{b} f(a+b-x) dx$

### 3. 华里士公式 (Wallis 公式)
用于快速计算 $\sin^n x$ 或 $\cos^n x$ 的定积分：

- **(5) 基础形式**:
  $I_n = \int_{0}^{\frac{\pi}{2}} \sin^n x dx = \int_{0}^{\frac{\pi}{2}} \cos^n x dx = \begin{cases} \frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{2}{3} \cdot 1, & n \text{ 为大于 1 的正奇数} \\ \frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{1}{2} \cdot \frac{\pi}{2}, & n \text{ 为正偶数} \end{cases}$

- **(6) $\int_{0}^{\pi}$ 形式**:
  $\int_{0}^{\pi} \sin^n x dx = 2 \int_{0}^{\frac{\pi}{2}} \sin^n x dx$
  $\int_{0}^{\pi} \cos^n x dx = \begin{cases} 0, & n \text{ 为正奇数} \\ 2 \int_{0}^{\frac{\pi}{2}} \cos^n x dx, & n \text{ 为正偶数} \end{cases}$

- **(7) $\int_{0}^{2\pi}$ 形式**:
  $\int_{0}^{2\pi} \sin^n x dx = \int_{0}^{2\pi} \cos^n x dx = \begin{cases} 0, & n \text{ 为正奇数} \\ 4 \int_{0}^{\frac{\pi}{2}} \sin^n x dx, & n \text{ 为正偶数} \end{cases}$
